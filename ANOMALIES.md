# Hearthstone Battlegrounds — Anomalies Reference

Anomalies are game-wide modifiers shown to all players before hero selection.
One anomaly is active per game (randomly chosen from the active pool).

**Current status (as of Season 13 / April 2026):** Anomalies were removed in
Season 12 (December 2025) and replaced by the Trinket system. They were last
active in **Season 11 (Patch 34.6)**.

---

## How Anomalies Work

- Revealed before hero selection — all 8 players share the same anomaly.
- Some anomalies ban specific heroes or minion types.
- Anomalies are weighted; not all appear at equal frequency.

## Simulator Implementation Notes

Most anomalies affect game-level rules rather than individual card mechanics, so
they require handling in `env/game_loop.py` rather than `symbolic/effect_handler.py`:

| Category | Where to implement |
|---|---|
| Economy tweaks (Grapnel, Eleventh Hour, Golden Arrow) | `step_shopping` — modify gold/cost logic |
| Golden / triple modifiers (False Idols, Perfected Alchemy) | `env/triple_system.py` + `_cast_spell` |
| Starting board (Scout's Honor) | `reset()` — place minion on board at game start |
| Hero power pool (Cosmic Duality) | `reset()` — double hero power selection |
| Shared offers (Audience's Choice) | Requires multi-player coordination in game loop |
| Tier 7 (Secrets of Norgannon) | `tavern_tier` cap: allow 7; add T7 cards to pool |
| Timewarped Tavern anomalies | Require full Timewarp system (not yet implemented) |
| Buddies (Bring in the Buddies) | Requires Buddy system (not yet implemented) |

**Not implemented:** Timewarped Tavern anomalies and Buddy anomalies depend on
mechanics (Chronum currency, Buddy buttons) that are not yet modeled in the
simulator. These are tracked here for future reference.
